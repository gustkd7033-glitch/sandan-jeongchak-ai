## 서비스 소개

전국 산업단지에서 일하는 외국인 근로자(E9 비자)가 언어 장벽 없이 지원센터·병원·은행·출입국 정보를 즉시 확인할 수 있는 RAG 기반 다국어 AI 챗봇입니다.

**"검색은 어렵고, 통역은 비싸다."**
산단 정착 AI는 검증된 공공데이터만을 근거로 정확하게 답변하여 잘못된 정보로 인한 피해를 방지합니다.

---

## 주요 기능

| 기능 | 설명 |
|---|---|
| 🌐 **7개 언어 완전 다국어화** | 한국어·베트남어·네팔어·인도네시아어·미얀마어·태국어·영어 — UI 전체 + 챗봇 답변 |
| 🏭 **산업단지 선택형 위치 기반** | 6개 산단 선택 시 해당 지역 정보 자동 필터링 |
| 📍 **답변 연동 인라인 지도** | 챗봇 답변 후 관련 기관 위치가 지도에 자동 표시 |
| 🗺️ **전국 산단 현황 지도** | 22개 외국인전용 산업단지 + 지원센터 통합 지도 |
| 📊 **E9 통계 기반 인사이트** | 모든 답변에 지역별 E9 근로자 통계 자동 첨부 |
| 🛡️ **RAG 기반 환각 방지** | 공공데이터에 없는 내용은 추측 없이 기관 직접 문의 안내 |

## 기술 스택

```
Frontend  Streamlit (Python)
AI        Google Gemini API (gemini-3-flash-preview)
RAG       직접 구축한 knowledge_base.json 기반 키워드 검색 + 컨텍스트 주입
지도      Folium + streamlit-folium (OpenStreetMap)
배포      Streamlit Cloud
데이터    법무부 체류외국인 통계 / 각 지자체 외국인지원센터 공식 안내 /
          근로복지공단 / 직접 조사한 전국 22개 외국인전용 산단 좌표
```

## 로컬 실행 방법

```bash
# 1. 패키지 설치
pip install streamlit google-generativeai folium streamlit-folium

# 2. secrets 설정
mkdir -p .streamlit
echo 'GEMINI_API_KEY = "your_key_here"' > .streamlit/secrets.toml

# 3. 실행
streamlit run app_gemini_v2.py
```
